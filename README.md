# 📌 项目标题

> 基于 React + WebGL 的个人展示网站

---

## 🖼 项目简介

这是一个使用 React 构建的个人展示网站，结合 Three.js 和 Canvas 动态背景，展示个人信息、技能、经历等内容。网站支持响应式设计，兼容 PC 与移动端，具备丰富的交互动画与 3D 视觉效果。

---

## 🚀 技术栈

- **前端框架**：React + Vite
- **3D 渲染**：Three.js
- **动画**：Canvas、CSS Animation
- **适配**：响应式布局
